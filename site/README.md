# Laplace Analytica Landing Page

Este é o site da agência Laplace Analytica, desenvolvido com HTML, CSS e JavaScript modernos.

## Como abrir

Basta abrir o arquivo `index.html` no seu navegador preferido (Chrome, Safari, Firefox, etc.).

## Estrutura

- `index.html`: Estrutura principal do site.
- `styles.css`: Estilos visuais (tema roxo/preto/branco, animações, responsividade).
- `script.js`: Lógica para o menu mobile e formulário.
- `logo.png`: Logo da empresa.

## Funcionalidades

- Design responsivo (funciona em mobile e desktop).
- Formulário de captura de leads com os campos solicitados.
- Animações suaves e efeitos de "glow".
- Seções de Serviços, Sobre, Projetos e Contato.
