# Carrosséis — Prompts de Imagem
## Laplace Analytica Cognitive

> Fluxo de trabalho:
> 1. Você gera a imagem do prompt no Midjourney, DALL-E 3 ou similar
> 2. Manda a imagem pra mim
> 3. Eu construo o carrossel em HTML com a imagem de fundo + texto
> 4. Você exporta como PNG

> Fundo escuro | Texto branco | Destaque roxo #7c3aed | Fonte Outfit

## Como gerar as imagens

**DALL-E 3 (ChatGPT Plus):** Cole o prompt diretamente. Funciona bem pra todos.
**Midjourney:** Cole o prompt e adicione `--ar 4:5 --style raw` no final.
**Leonardo AI:** gratuito com créditos diários, boa qualidade.

Depois de gerar, manda a imagem aqui e eu monto o carrossel completo em HTML pra você exportar.
